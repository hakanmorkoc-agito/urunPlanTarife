# Public Klasörü

Bu klasöre logo görselini ekleyin:

- **Dosya adı:** `logo-a.png`
- **Konum:** `public/logo-a.png`

Logo görseli mor arka plan üzerinde beyaz "a" harfi içermelidir.

Eğer görsel farklı bir formattaysa (jpg, svg vb.), Sidebar.jsx dosyasındaki görsel yolu ve dosya adını güncelleyin.
